Get started

Files

Once downloaded, unzip the compressed folder to see the structure of Flatty template.

What files are required for minimal setup?

Stylesheets:
bootstrap/bootstrap.css , light-theme.css or dark-theme.css or dark-blue-theme.css , theme-colors.css (optional) .

Javascripts:
jquery/jquery.min.js , jquery/jquery-migrate.min.js , bootstrap/bootstrap.min.js , theme.js .

Loading other, than required files (plugins, jQuery UI, etc) is based on you needs.

How to get plugin ABC get working?

If you want plugin ABC to get working, you must include all dependent files - check all folders stylesheets/plugins/ABC/ , javascripts/plugins/ABC/ , images/plugins/ABC/ .

If you are not familiar with Compass or SASS, check these sites:

How can I benefit from Compass & SASS?

Flatty was built using SASS, which is an CSS3 extension with variables, mixins, selector inheritance, etc. Most of custom work in template can be overriden by usage of variables. All assets in stylesheets are linked with usage of url/path helpers from Compass.

SASS

How to change Bootstrap variables?

You have possibility to compile/override your own version of Bootstrap. But it's not necessary - future updates will be easier with standard Bootstrap (Flatty is using standard Bootstrap and NOT custom Bootstrap).

For example if you want to change base font size of Bootstrap from 14px to 13px. Open file compass/sass/bootstrap/bootstrap.scss and override Bootstrap variable by adding variable with same name to this file: $baseFontSize: 13px; .

How to change Flatty variables?

Variables in Flatty are dependent on theme. Let's suppose that you choosed light-theme.scss as your theme. Then open that file. For example, you want to change body background color. And background color is now red :-)

I don't want to include all theme colors, but just one. How can I do that?

If you don't want ability to change themes by using contrast-red, contrast-blue, ... classes on body tag then it's not necessary to load theme-colors.scss file.

Changing contrast color is as simple as going to your theme file ( light-theme.scss for example) and overriding $contrast variable. And theme color is navy blue, now.

If you are not loading theme-colors.scss be sure that you are importing @import "theme/variables"; on the first line of your custom build.


Ruby on Rails

Flatty is Ruby on Rails ready! So implementation couldn't be easier :-)

How to install Flatty?

  1. Just go to flatty/ruby_on_rails folder and copy assets folder to your_rails_app/app
  2. Copy application.js from flatty/ruby_on_rails/pipeline to your_rails_app/app/assets/javascripts
  3. Copy application.scss from flatty/ruby_on_rails/pipeline to your_rails_app/app/assets/stylesheets
  4. Add compass-rails gem to your Gemfile. gem "compass-rails", "~> 2.0.alpha.0"
  5. You're done!

Custom application.js & application.scss

Flatty is sent with application.js & application.scss files. These files are loading all Flatty files (these asset pipeline files are just examples). So you can create your own build as described above in Compass & SASS section.

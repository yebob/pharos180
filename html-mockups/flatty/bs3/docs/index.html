What to expect from this template?

Flatty is fully responsive template used to make your administration more beautiful. Flat design is the latest trend in webdesign and you find all beauties of this trend in this template! Difficult things in your application will look more easier thanks to Flatty.

Flatty is suitable for mobile phones, as well as for tablets and other devices, so it satisfies any kind of user. The advantage of this template is that you don't have to deal with look of your application and you can focus on more important things.


What do you need to know?

Flatty is built on Twitter Bootstrap framework. If you are not familiar with Bootstrap, you should check it out because this documentation expects a basic knowledge of this framework.
This documentation will not teach you HTML & CSS, it's just showcase of custom elements used in template. The basic things from Bootstrap aren't described (documentation for these things can be found here)
